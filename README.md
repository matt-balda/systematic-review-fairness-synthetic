# Systematic review: Fairness & Bias, Synthetic Data and IA in Clinical Contexts

## Objective

This repository contains all materials for a systematic review on the generation of synthetic data aimed at promoting fairness and mitigating biases in underrepresented groups, applied to AI in clinical contexts.
